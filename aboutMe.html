Quiénes somos

Somos Arnau Aumedes y Alex Ruiz, estudiantes de 2º de Desarrollo de Aplicaciones Web (2DAW). Actualmente estamos realizando prácticas y queremos dar el salto al mundo profesional de la programación. Nos une la curiosidad, las ganas de aprender y la intención de crear proyectos útiles y bien construidos.

Perfil de Arnau

Me interesa el desarrollo front-end y mejorar la experiencia del usuario. Disfruto maquetando interfaces claras y aplicando buenas prácticas de semántica, accesibilidad y estructura. Me motiva aprender herramientas modernas y trabajar en equipo.

Perfil de Alex

Me atrae la lógica del back-end y cómo las piezas encajan detrás de una aplicación. Me gusta escribir código organizado, entender cómo fluye la información y aprender sobre bases de datos y APIs. También valoro la colaboración y el versionado correcto.

Habilidades que estamos desarrollando

  • HTML y CSS para crear interfaces limpias y adaptables.
  • JavaScript para añadir interacción y lógica básica.
  • Git y control de versiones.
  • Organización y constancia en el trabajo diario.
  • Aprendizaje autónomo y práctica continua.

Nuestro enfoque

Buscamos aprender haciendo: planificamos una idea sencilla, la desarrollamos paso a paso y luego la mejoramos. Tratamos de mantener el código claro, comentar lo necesario y evitar complicaciones antes de tiempo. Valoramos el apoyo mutuo y la revisión entre compañeros.

Objetivos actuales

  • Consolidar bases sólidas en HTML, CSS y JavaScript.
  • Mejorar el uso de Git y trabajo colaborativo.
  • Crear pequeños proyectos que muestren nuestro progreso.
  • Aprender fundamentos de back-end y bases de datos.
  • Prepararnos para oportunidades laborales en desarrollo web.

Lo que nos motiva

“Aprender un poco cada día y ver cómo algo que no existía empieza a funcionar gracias a nuestro trabajo.”
